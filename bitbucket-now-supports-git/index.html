BitBucket now supports Git!

Watch out GitHub, for the one year anniversary of Bitbucket joining Atlassian, they’re finally announcing Git support!

They’ll be the GitHub diehards, that say ‘GitHub’ is still infinitely better, but I’d argue that Bitbucket’s not that far behind, and now with the addition of Git support, you can keep all of your code in one place for the first time.

To aid the transition, you can import your existing Git, Mercurial and Subversion sourcecode, and to nab as many GitHub customers as possible, they’ve even added a GitHub importer – that’s nice of them :)
