No Java installed after OSX Lion upgrade

I haven’t opened IntelliJ IDEA for a while, and was completely flabbergasted to be told that I had no Java installed when trying to do so!

I use Java quite regularly, so was dismayed to be told it wasn’t installed. It dawned on me that I’d recently upgraded to OSX Lion, and this is where the culprit lay. The upgrade process left intact all of my other system files, so why the hell not my Java installation???

So, here we go again – a reinstall of an OSX compatible Java kit.
