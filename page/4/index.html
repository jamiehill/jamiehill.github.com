Adobe AIR 3 and the new Native Extension capabilities

Adobe’s latest incarnation of the AIR runtime – Adobe AIR 3 – marks a staggering leap forward in the platforms capabilities, and opens the door to the future of Flash as an RIA development and deployment solution, on the desktop and mobile.

One of the biggest wins us developers have gained, is the ability to actively extend the runtime’s core capabilities using the new Native Extension API.

The native extensions in a nut shell, couple Actionscript classes with native code that provides easy access to device specific libraries and features that previously we could only dream about.

Want to initiate an Android phone’s vibration from AIR? No problem. Want to read pressure sensitivity data from a pen tablet? No problem! The possibilities are endless!

More over, rather than just having a new all singing, all dancing feature set, it offers the possibility of providing significantly deeper integration with target devices, legacy code reuse and huge performance gains.
